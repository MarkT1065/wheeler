About Polygon.io API

The Polygon.io API provides real-time and historical market data for stocks, options, and other financial instruments.

  • Free Tier: 5 API calls per minute, delayed data
  • Starter Plan: 100 API calls per minute, real-time data
  • Developer Plan: 1000 API calls per minute, additional endpoints

Your API key will be used to automatically update stock prices and fetch dividend information.
